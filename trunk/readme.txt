============================================================
 1. インストール方法
============================================================

インストーラーはありません。
blue-sky ディレクトリをローカルにコピーして下さい。


============================================================
 2. ゲームの始め方
============================================================

ローカルにコピーした blue-sky ディレクトリの中の
blue-sky.exe を実行して下さい。


============================================================
 3. アンインストール方法
============================================================

ローカルの blue-sky ディレクトリをまるごと削除して下さい。
レジストリは使用していません。


============================================================
 4. ディレクトリ構成
============================================================

/
	blue-sky/
		blue-sky.exe					ゲームの実行ファイルです。
		blue-sky.config					ゲームの設定ファイルです。
		
		log/							ログが出力されるディレクトリです。
		manual/							操作マニュアルです。
		media/							ゲームで使う素材ファイルが入っています。
			image/							画像
			model/							3D モデルデータ
			music/							音楽
			object/							建物データ
			shader/							シェーダー
			sound/							効果音
			stage/							ステージデータ
			stage_ext/						追加ステージデータ
		media_source/					素材の編集用データです。Blender と GIMP のファイルが入っています。
		save/							セーブデータが入っています。
		spec/							ゲームの仕様が入っています。書きかけです。
		src/							C++ ソースコードです。プログラムの勉強にどうぞ。
		
	tokyo-shojo.mp3						テーマソングの MP3 ファイルです。


============================================================
 5. 使用ライブラリ
============================================================

このプログラムは ogg vorbis ライブラリを使用しています。
